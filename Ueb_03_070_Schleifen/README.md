# Programmierübung: Schleifensteuerung in ST

Gegeben ist ein Programm, das aktuell alle Werte eines Arrays aufsummiert.

## Aufgabe

Erweitere das Programm schrittweise:

---

### Teilaufgabe 1: Negative Werte ignorieren (CONTINUE)
Passe die FOR-Schleife so an, dass **negative Werte nicht in die Summe eingehen**. Verwende dazu die Anweisung `CONTINUE`.

---

### Teilaufgabe 2: Früher Abbruch (EXIT)
Ergänze die Schleife so, dass sie **abgebrochen wird, sobald ein Wert größer als 80 gefunden wird**. Verwende dazu `EXIT`.

---

### Teilaufgabe 3: Zusätzliche Auswertung
Erweitere das Programm um eine Variable `nCount`, die zählt,  
wie viele Werte tatsächlich zur Summe beigetragen haben (also nicht übersprungen wurden).
